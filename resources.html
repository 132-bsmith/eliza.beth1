Brackets

Brackets is an open-source code editing program that is designed to make writing and editing easy. The live preview option allows you to view the changes you make to your browser page in real time, and itâ€™s one of the biggest reasons I chose to use Brackets instead of Notepad++ for my second try at coding despite being more familiar with the latter.

I found in general that Brackets is more user friendly. It is true that I understood web design better when starting this project than I did when starting the first one, but it has been much easier for me to navigate and trobleshoot on my own. Even saving and loading files was simpler with the two file location sidebars; one for all of the files in my webpage folder and the other called "Working Files", for files that were in the editing process.

W3Schools and Bootstrap

I used W3Schools and Bootstrap as a source for the majority of my code. The tutorials on both sites were extremely helpful to a newbie like myself. I used W3Schools the most, but I must credit Bootstrap with the code that started my navigation bar. I customized it with different colours, texts and spacing to fit the layout of my webpage, but it provided me with the structure I had been struggling to write from scratch.

W3Schools provided me with both html code and css, which I used for everything on my webpage. I stuck mostly to the html tutorial page for colours, fonts, links, images, etc. but added in some css when I wanted a more complex page layout. For that I used a variety of different column layouts as well as some float and align properties.
