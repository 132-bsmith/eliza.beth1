My project is…

I chose to create this website for my final Computer Studies project because I wanted to go back and revisit the topic I felt I had made the most progress in during the course, which was designing my first webpage with Notepad++. I even named my file folder "My Second Webpage" to match the "My First Webpage" folder from September of last year.

Here is the link to my first webpage for comparison.

I was/am curious about this project because…

Web design was the subject in the course that was most unfamiliar to me, so I was curious about things I had discovered during the initial web design unit that I hadn't been able to explore properly. I feel much more adept now that I've had more time to play and experiment.

So far I have learned and completed…

In this webpage I demonstrated most of the skills I learned and added to them. I learned a lot of new element tags, and how they work. For instance, I don't believe I used <div> at all in my first webpage, and if I did I just copied and pasted it into where I wanted it to go. For this webpage I wrote and edited most of my own <div> elements and learned how to use classes. I also learned more about general code writing do's and don'ts, for instance when I started the project, I had a separate style element for each main section of my page. After having some difficulties getting a piece of code to work, I ended up reorganizing so that all of my style elements were together as one element.

Some problems, iterations and/or hurdles have been…

I started this project with no clue about what I wanted to put on my webpage, so I went through quite a few different iterations before settling on using it as my presentation about itself. I didn't have many problems, I think the biggest mistake I made was the day I forgot that I have to manually save my files and accidentally deleted a days work. I made very sure it didn't happen again. Another interesting hurdle was when I put a link to my first webpage onto this one. I tested the page before I did so to make sure everything was working, but as soon as I tried it via the link, it stopped working. It took me a lot of fiddling before I realized I just had to copy all of the files inbot the right folder so the files were findeable.

My Key Learnings to date are…

  1. A better understanding of html as a language
  2. Code organization
  3. User interface vs system interface
